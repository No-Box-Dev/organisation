# No Box Dev Branding

This folder contains the official branding assets and guidelines for No Box Dev.

## Brand Identity

### Tagline

> **"Shaped by psychology, enhanced by AI."**

### Core Message

We build web and mobile applications that don't just function — but make sense to the people using them.

### Brand Values

- **Human-centered** — We focus on how people think, decide, hesitate, and act
- **Clarity** — No vague promises, clear communication and shared responsibility
- **Collaboration** — Hands-on, committed, and closely involved partnerships
- **Quality** — We don't ship half-finished work

## Logo Color Variants

Font files are in the `fonts/` folder:
